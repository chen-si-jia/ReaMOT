## 🔍 ReaMOT vs RMOT

![](asset/RMOT_vs_ReaMOT.jpg)

**Comparison between the traditional RMOT and the proposed ReaMOT tasks.**

(a) The traditional **RMOT** task relies on explicit visual attribute matching (e.g., "car", "right"). Consequently, existing models designed for this standard paradigm fail when confronted with implicit language instructions. 

(b) In contrast, the **ReaMOT** task elevates the tracking process to a cognitive level. It requires models to infer and track the specific targets that satisfy implicit semantic constraints (e.g., "attacking players") through multiple steps of logical reasoning.
